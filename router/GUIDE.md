# NITOS на роутере — простой гайд

VPN ставится **один раз на роутер** — и дальше работает у всех устройств дома
сразу: телефоны, ноутбуки, телевизор, приставки. На них ничего ставить не надо.

## Подойдёт ли мой роутер?

| Роутер | Подойдёт? | Что делать |
|---|---|---|
| **Keenetic** | ✅ Да | [Гайд для Keenetic](#keenetic) — без перепрошивки |
| **GL.iNet** | ✅ Да | [Гайд для GL.iNet](#glinet) — работает из коробки |
| **Xiaomi / Redmi, TP-Link, ASUS, Netgear…** | ⚙️ Да, после прошивки | [Гайд через OpenWRT](#openwrt) |
| **Huawei, Tenda, роутер от провайдера** | ❌ Нет | Прошивка закрыта. Проще купить б/у Keenetic |

> Роутер должен быть не слишком старый: от 128 МБ памяти. На совсем слабых
> скорость через VPN будет низкой.

<a id="keenetic"></a>
## Keenetic (без перепрошивки)

Понадобится USB-флешка (любая, от 1 ГБ — всё на ней сотрётся).

1. Откройте админку роутера: `my.keenetic.net`
2. **Общие настройки → Изменить набор компонентов** → включите
   **«Поддержка открытых пакетов (OPKG)»** и дождитесь установки.
3. Вставьте флешку. В разделе **«Накопители»** отформатируйте её в **ext4**.
4. Поставьте **Entware** по официальной инструкции Keenetic:
   откройте `help.keenetic.com` и найдите в поиске **«Entware»**
   (кратко: скачать установщик под вашу модель → положить на флешку в папку
   `install` → в админке **Приложения → OPKG** выбрать флешку).
5. Подключитесь к роутеру по SSH **на порт 222**
   (Windows: программа PuTTY; логин `root`, пароль `keenetic`):
   ```
   ssh -p 222 root@192.168.1.1
   ```
6. Выполните [команду установки](#install) — готово.

<a id="glinet"></a>
## GL.iNet (самый простой)

1. Подключитесь по SSH: логин `root`, пароль — тот же, что от админки:
   ```
   ssh root@192.168.8.1
   ```
2. Выполните [команду установки](#install) — всё.

<a id="openwrt"></a>
## Xiaomi, TP-Link и другие (через OpenWRT)

Родная прошивка этих роутеров закрыта — сначала ставится свободная прошивка
**OpenWRT** (один раз, минут 15).

1. Проверьте, что ваша модель поддерживается: `firmware-selector.openwrt.org`
   (введите модель роутера).
2. Прошейте по инструкции для вашей модели с `openwrt.org` (обычно: скачать
   образ → загрузить в роутер через его же веб-интерфейс как «обновление»).
   ⚠️ Внимательно читайте гайд именно вашей модели.
3. Подключитесь по SSH:
   ```
   ssh root@192.168.1.1
   ```
4. Выполните [команду установки](#install).

<a id="install"></a>
## Команда установки

Скопируйте ссылку вашей подписки в боте **@nitos_vpn_bot** и выполните на
роутере (вставьте свою ссылку в кавычки):

```sh
curl -fsSL https://raw.githubusercontent.com/Heekuez/nitos-releases/main/router/install.sh | sh -s -- "ВАША_ССЫЛКА"
```

Скрипт сам поставит движок, проверит конфиг, поднимет туннель и покажет новый
внешний IP. Вся домашняя сеть — через VPN.

**Удаление** — та же команда, только с `--uninstall` вместо ссылки.

## Если не заработало

- OpenWRT/GL.iNet — посмотрите лог: `logread | grep sing-box`
- Keenetic — лог: `/opt/var/log/nitos.log`
- Не разобрались — напишите в поддержку: **@nitos_vpn_bot**
